## Dependencies

- Node.js
- Express
- EJS
- bcryptjs
- cookie-session

# devDependencies

- chai
- mocha
- nodemon


## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `npm start` command.
- Run the unit test using the `npm run test` command.
